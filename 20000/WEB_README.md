# 20000 — 웹 브라우저 플레이 버전

브라우저에서 바로 플레이할 수 있는 HTML5 버전입니다.

## 실행 방법

1. **로컬 실행**  
   `index.html` 파일을 더블클릭하거나 브라우저에서 열기.

2. **로컬 서버** (권장 — 일부 브라우저 보안 정책 회피)  
   ```bash
   cd 20000
   python -m http.server 8080
   ```
   브라우저에서 http://localhost:8080 접속.

## 조작

- **PC**: 방향키(←→↑↓) + 스페이스(발사)  
  게임 오버/클리어 시 R(재시작), M(메인 메뉴), ESC(종료)
- **모바일/태블릿**: 화면 하단 가상 버튼 (방향 + FIRE)

## 저장

- Best Rank, 보스 클리어 기록은 **LocalStorage**에 저장됩니다.
- 같은 브라우저에서 다시 접속해도 유지됩니다.

## 게임 내용

- Pygame 원본과 동일: 4단계 보스(Bronze → Silver → Gold → Platinum)
- 5% 확률 파란 회복탄(체력 30 회복)
- 업그레이드 게이지(적 처치 시 충전)
- 플래티넘 보스 격파 시 게임 클리어
